# 本地AI聊天助手

本地AI聊天助手是一个基于Next.js和React开发的Web应用，可以连接到本地运行的Ollama服务，让你与各种大型语言模型进行对话交流，无需依赖云服务。

## 主要特点

- **本地运行**: 所有AI处理都在本地完成，保护隐私和数据安全
- **多模型支持**: 集成llama3.2、deepseek-coder-v2等多种AI模型
- **助手市场**: 内置多种特定领域的AI助手，如代码专家、写作助手等
- **自定义助手**: 可创建自定义角色和特定场景的AI助手
- **流式响应**: 实时展示AI生成内容，提供更自然的对话体验
- **会话管理**: 创建、保存和管理多个对话会话
- **文件上传**: 支持上传文件供AI分析和处理
- **持久化存储**: 自动保存所有会话历史和设置

## 技术栈

- **前端框架**: Next.js 14 (React)
- **UI组件**: Tailwind CSS, Shadcn UI
- **状态管理**: Zustand with Immer
- **AI集成**: Ollama API
- **响应式设计**: 适配桌面和移动设备
