# 对话管理

对话管理是本应用的核心功能，允许用户创建、存储和管理多个聊天会话。

## 主要功能

### 创建新会话

点击侧边栏顶部的"新建对话"按钮可以创建一个新的聊天会话。你也可以通过选择一个助手来创建特定领域的对话。

### 会话列表

侧边栏的"历史对话"部分显示所有已保存的会话，按照最近更新时间排序。每个会话项显示:
- 会话标题
- 最后更新时间
- 删除按钮(鼠标悬停时显示)

### 会话内容

每个会话包含:
- AI和用户之间的消息历史
- 会话使用的AI模型信息
- 如果使用了特定助手，会显示助手名称和图标

### 删除会话

将鼠标悬停在会话项上会显示删除按钮，点击后会请求确认，确认后会从存储中永久删除该会话。

## 技术实现

会话管理通过Zustand状态管理库实现，具有以下特点:
- 使用Immer进行不可变状态更新
- 自动持久化到localStorage
- 响应式更新UI

会话数据结构:
```typescript
interface ChatSession {
  id: string;
  title: string;
  createdAt: Date;
  updatedAt: Date;
  messages: Message[];
  assistantId?: string;
}
```
