# 助手功能介绍

在本页面中，您将了解如何使用助手功能来提升您的聊天体验。助手可以帮助您完成各种任务，从回答问题到提供建议。

## 什么是助手？

助手是一个智能工具，能够根据您的输入提供实时反馈和建议。您可以与助手进行对话，获取信息或执行特定任务。

## 如何使用助手？

1. **选择助手**：在聊天界面的侧边栏中，您可以选择不同的助手。每个助手都有其独特的功能和特点。
   
2. **发送消息**：在输入框中输入您的问题或请求，然后点击发送按钮。助手会根据您的输入进行响应。

3. **查看响应**：助手的响应将显示在聊天窗口中。您可以继续与助手对话，获取更多信息。

## 助手的功能

- **信息查询**：助手可以回答您的问题，提供相关信息。
- **任务执行**：您可以请求助手执行特定任务，例如设置提醒或发送文件。
- **个性化建议**：助手可以根据您的偏好提供个性化的建议。

## 示例

以下是与助手对话的示例：

用户: 你能帮我查一下天气吗？
助手: 当然可以！请告诉我您想查询的城市。


## 常见问题

### 助手可以处理哪些类型的请求？

助手可以处理多种请求，包括信息查询、任务执行和个性化建议。

### 如何添加自定义助手？

您可以在聊天界面的助手市场中添加自定义助手。只需填写助手的名称、描述和提示词即可。

## 结论

助手功能为您的聊天体验增添了更多可能性。通过与助手的互动，您可以更高效地获取信息和完成任务。
